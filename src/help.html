Can't get Runner to work.

I'm Still running OS X Yosemite (10.10)

There were some issues on previous versions, but now we support Yosemite starting with version 0.6 of the plugin. If you keep experiencing this issue please let us know.


I am using a third-party unarchiver

There is also a known issue when unarchiving the .zip file with a third-party tool (e.g. "The Unarchiver"). Please try with the native OS X unarchiver (Archive Utility) and let us know if it's working!

We unfortunately can't grab all menu items due to some restrcitions in Sketch. Let us know which ones are missing!

This should be fixed with version 0.6 of the plugin. If you keep experiencing this issue please report it.
